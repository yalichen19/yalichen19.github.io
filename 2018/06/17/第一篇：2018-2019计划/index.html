第一篇：2018-2019计划

受到撒网要见鱼博客的启发，花了一些时间搭建了自己的博客，主要是记录一些技术的知识点以及自己的计划和感想。不确定能写多少，但起码要有一个好的开始。

马上，我就要大学毕业了，确定了工作，也租好了房子，就要开始新的生活。古典的《拆掉思维里的墙》中提到过我们应该常常对自己发问：“你到底想成为一个怎样的人”。这也是我最近常在思考的问题，我到底想变成什么样人，一年之后有怎样的进步？我不想浑浑噩噩地度过每一天，我希望我的时间是有规划的，于是我把这篇文章作为自己博客的第一篇，当作是对自己的认真回答。

技术学习

前端技术

目标1：巩固已学的知识（js闭包、原型链、设计模式）

学习前端从去年9月份到现在，其实学得十分狼狈，边学习边从秋招面试到了春招，很多知识还来不及完全消化，所以基础知识还需要再复习巩固。这部分主要是通过书和渡一教育的课程。

目标2：学习React和React相关的全家桶

之前学习的都是Vue，好不容易对Vue使用熟练了，对底层的状态存储，设计模式有些许了解了，结果公司用的是React，那就只好再学吧。计划先从公司的需求和个人的小项目开始，使用熟练后再去使用React-router、React-Native等，之后再去了解下源码和底层实现。

目标3：对公司项目代码有更深刻的认识

实习了快两个月了，但自己还是停留在修改代码，实现需求的水平。对于公司项目的代码的整体实现不够清晰。这一点需要结合下一个目标一起实现，需要自己尝试实现整个项目。

目标4：学习自己搭建整个项目

对webpack基本使用有了解，每次创建项目都是用脚手架生成的，因此对整个项目的搭建还不够清晰。需要从零搭建起一个项目，并且整合多个工具的使用。

目标5：解决面试中遇到的那些还不懂的问题

这里需要顺便谈一谈面试的体会，太痛苦了，自己大大小小公司面试了估计总共有20家吧。痛苦的点在于，很多公司不是理想的，但为了工作还是需要去面试，最后人家也并不收你。
还好现在自己有了工作，尽管还有不完美，但自己更多的是感激，同时也很感激每一次的面试，给我提高表达能力和认识自己的机会。还有很多面试的问题，也成为我学习的方向，比如：

  1. 从输入URL到页面加载的过程？如何由一道题完善自己的前端知识体系！
  2. new Function 和eval执行引入js代码字符串，选择使用哪一种，为什么？
  3. z-index是绝对的吗？，z-index栈是什么
    …
    这些问题，希望之后我都能够回答。

网络知识

目标1：学习网络的知识，包括前端涉及的http请求、局域网路由等等

现在回忆一下《计算机网络》课程学的东西，真是完全不理解，我想这是中国教育的通病，考试考细节内容，于是上课的重点也是一些细节和计算。对于网络知识的整体没有梳理过，课本每一章的划分的关联也不懂。
抱怨的同时其实也是对自己的反思，没有主动去解决，自主学习。所以现在还是需要再回顾学习，目前想到的方法还是先看书和检索吧，如果不够，再寻找其他途径。

目标2：自己搭建服务器翻墙

目前已经了解了一些翻墙的方式，但其中的原理还不是很清楚。希望自己可以亲手实现一下。

服务器及其他

目标1：学习node

因为毕设对node有了了解，之后还需要再深入学习一下。我的老大要交给我脚手架工具的维护，所以对node的掌握还是很必要的。

目标2：提高专业素养

所谓专业素养，其实就是多了解互联网、前端的常识，增加知识广度和谈资，保持信息的及时更新。这点需要关注微信一些公众号推送文章，技术博客、github热门项目等。

英语学习

目标1：提高口语

一直以来自己英语成绩都不错，然而发音和口语挺灾难的，每次遇到有和老外交流的突发状况，基本都蹦不出几个单词。不过，有一次还是帮一个小姐姐从小哥那里领到了快递，心里也是很开心。总之，先跟着新概念的课程看书看视频，多说多背。

目标2：提高词汇和阅读

真的需要强迫自己阅读英文文献，也要学着看英语原版书。整理大学课本的时候特意留下了《Reader’s choice》，自己怎么也是曾经英语A班的，忘记的内容都要一点点捡起来。

阅读

目标1：一个月3本书，一年保30本争40本

犹太人平均一年读50本书，中国人平均不到1本…
多读书，从我做起。目前的阅读书单主要是成哥推荐的《引爆点》、《异类》、《未来简史》、《人类简史》、《乌合之众》、《耶路撒冷三千年》、《失控》、《必然》，之后再加。

目标2：写摘要和读后感

读完一本书，对我这种记忆力不太好的，可能时间长了并不会有什么影响。所以多记录、多书写，之后的读书笔记也会放在自己的博客上。

运动

目标1：一周有一次锻炼

我知道把运动放在最后，说明确实没有太看重，但是还是要强迫自己重视锻炼，毕竟本来自己的身体就不是很好，希望能先把体重增上去。也希望我的新任室友能够带动我一起去健身房。

总结

其实除了以上，还想提高一下自己的艺术修养，包括绘画、弹吉他，我还喜欢做手工，但这些就不一一列举了。计划总是很丰满，但愿一年之后，能够完成80%，也算是有所收获了。不会要求自己有多少成就，单是享受上进的过程，加油。
